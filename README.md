# Paws & Preferences

A fun single-page web application that lets users discover their favorite cats or kittens by swiping through a stack of cat pictures—similar to popular dating apps.

---

## **Project Overview**

- Users can **swipe right** to like a cat and **swipe left** to dislike.  
- Works on **mobile (touch swipe)**, **desktop (mouse drag)**, **buttons**, and **keyboard arrows** (← / →).  
- After swiping through all cats, a **summary page** shows how many cats were liked and displays the images.  
- Cat pictures are sourced from [Cataas](https://cataas.com/).  
- Designed to be **mobile-friendly** with smooth animations.

- Mouse drag swipe for desktop  
- Buttons for desktop users  
- Keyboard arrow support  
- Animated card transitions  
- Summary page showing liked cats  
- Responsive and mobile-friendly design
